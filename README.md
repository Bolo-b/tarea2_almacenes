# Analisis de Personas Desaparecidas en Mexico (RNPDNO) - Equipo 1

Se aplico la metodologi­a **CRISP-DM** para realizar una limpieza profunda y un analisis exploratorio de los datos del Registro Nacional de Personas Desaparecidas y No Localizadas (corte 22-08-2023).

## Estructura del Proyecto
* `RNPDNO-22-08-2023.csv`: Dataset original asignado al Equipo 1.
* `limpieza_RNPDNO.ipynb`: **(Fase de Preparación)** Notebook reproducible con el proceso de limpieza, normalización y validación de datos.
* `EDA_desaparecidos.ipynb`: **(Fase de Modelado)** Notebook con el análisis estadístico, medidas de tendencia central, heterogeneidad y hallazgos.
* `requirements.txt`: Archivo de dependencias para asegurar la reproducibilidad del entorno.
* `_quarto.yml` & `index.qmd`: Archivos de configuración para la generación del **Quarto Book**.
## Instrucciones de Ejecucion
1. Clonar el repositorio.
2. Crear un entorno virtual(Linux): `python3 -m venv env`o en Windows: `python -m venv env` .
3. Activar el entorno(Linux): `source env/bin/activate`
Windows: `env\Scripts\activate` o  `\env\Scripts\Activate.ps1`.
4. Instalar dependencias: `pip install -r requirements.txt`.
5. Ejecutar los notebooks en orden:
    Para la limpieza y generacion del .csv limpio:
    `limpieza_RNPDNO.ipynb`
    (FALTAN ARCHIVOS)
